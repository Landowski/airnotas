Política de Privacidade


Bem-vindo à nossa página de política de privacidade. A sua privacidade é de extrema importância para nós. Nesta política, explicamos como tratamos seus dados ao utilizar o Airnotas, um serviço de notas online.

Coleta de Dados


Ao utilizar o Airnotas, a autenticação é realizada por meio do serviço do Google Firebase Auth, uma plataforma que permite o login de maneira segura e simplificada. Durante o processo de autenticação, o Google Firebase pode solicitar informações como email e senha, mas essas informações são gerenciadas diretamente pelo Firebase. Nós, do Airnotas, não coletamos, armazenamos ou acessamos esses dados pessoais de autenticação. Apenas utilizamos o Firebase Auth para garantir que você tenha acesso seguro às suas notas.
Não coletamos nenhum dado pessoal além do que é necessário para autenticação. O Airnotas salva apenas as notas de texto que você criar, e essas notas são vinculadas ao seu perfil autenticado no Firebase.

Armazenamento de Dados


As notas que você insere no Airnotas são armazenadas de forma segura na nuvem utilizando o Firebase Firestore. Isso permite que suas anotações sejam acessadas em qualquer dispositivo onde você faça login com sua conta. O Airnotas não armazena suas anotações localmente no dispositivo, exceto quando você está offline, e, nesse caso, o Firebase sincroniza os dados assim que a conexão com a internet é restabelecida.

Segurança


A segurança das suas informações é uma prioridade para nós. O Firebase oferece um alto nível de segurança para proteger seus dados de acesso não autorizado. No entanto, recomendamos que você não utilize o Airnotas para armazenar informações altamente sensíveis, como senhas ou dados financeiros.

Alterações nesta Política


Podemos atualizar esta política de privacidade periodicamente para refletir melhorias em nossos serviços ou mudanças na legislação. Qualquer modificação será publicada nesta página, por isso sugerimos que você a revise com regularidade.
